# دليل الاختبار اليدوي الشامل | Manual Testing Guide

## كيفية الاستخدام

1. شغل السيرفر: `npm run dev`
2. افتح المتصفح على `http://localhost:3000`
3. اتبع كل test case وحدد النتيجة (✅ نجح / ❌ فشل)
